---
title: 'Chains API (Beta)'
openapi: 'GET /beta/balance/chains'
---

The chains endpoint lists the supported blockchains for the Token Balances API.

<Note>
  - URL may change when we move from /beta/ to /v1/. This is planned in July 2024.
</Note>

# Column Descriptions

| Column                    | Description                                 | Type                |
|---------------------------|---------------------------------------------|---------------------|
| name                      | Canonical blockchain name                   | string              |
| id                        | chainid of blockchain                            | integer             |

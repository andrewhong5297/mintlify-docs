---
title: 'Insert'
openapi: 'POST /v1/table/{namespace}/{table_name}/insert'
---

<RequestExample>

```bash curl
curl --request POST \
  --url https://api.dune.com/api/v1/table/my_user/dataset_interest_rates/insert \
  --header 'X-DUNE-API-KEY: <x-dune-api-key>' \
  --header 'Content-Type: text/csv' \
  --upload-file @DGS10.csv # from https://fred.stlouisfed.org/series/DGS10
'
```

</RequestExample>

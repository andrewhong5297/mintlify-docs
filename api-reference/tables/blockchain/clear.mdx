---
title: 'Clear Data (beta)'
openapi: 'POST /v1/blockchain/{chain}/{table_name}/clear'
---

The `Clear` endpoint deletes all the data in the specified blockchain table, but does not drop the table.

<RequestExample>

```bash curl
curl --request POST \
  --url https://api.dune.com/api/v1/blockchain/my_chain/blocks/clear \
  --header 'X-DUNE-API-KEY: <x-dune-api-key>'
```


```python Python
import requests

url = "https://api.dune.com/api/v1/blockchain/my_chain/blocks/clear"

headers = {
    "X-DUNE-API-KEY": "<x-dune-api-key>"
}

response = requests.request("POST", url, data=data, headers=headers)
```

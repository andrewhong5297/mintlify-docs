## Table Description

The `transactions` table contains information about all transactions on the Berachain blockchain. Each row represents a single transaction and includes information such as block number, hash, timestamp, sender, recipient, value, gas, and gas price. Berachain's EVM-identical architecture ensures full compatibility with Ethereum tooling while leveraging its Proof-of-Liquidity (PoL) mechanism for enhanced efficiency.

---
title: 'Supported Chains'
openapi: 'GET /beta/transactions/chains'
---

The chains endpoint lists the supported blockchains for the Trnasactions endpoint.

<Note>
  - URL may change when we move from /beta/ to /v1/. This is planned in Q3 2024.
</Note>

# Column Descriptions

| Column                    | Description                                 | Type                |
|---------------------------|---------------------------------------------|---------------------|
| name                      | Canonical blockchain name                   | string              |
| id                        | Chainid of blockchain                       | integer             |

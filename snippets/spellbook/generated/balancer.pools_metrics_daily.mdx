## balancer.pools_metrics_daily 
This spell aggregates data from the trades, liquidity and protocol fees spells, by day and pool, while also displaying some basic information about the pool


<ParamField path="blockchain">
Blockchain
</ParamField>
<ParamField path="project">
Project name (balancer)
</ParamField>
<ParamField path="version">
Version of Balancer where the liquidity pool used in transaction is deployed
</ParamField>
<ParamField path="block_date">
Block date in UTC
</ParamField>
<ParamField path="project_contract_address">
Pool address
</ParamField>
<ParamField path="pool_symbol">
Token symbols, followed by their respective weights, if applicable
</ParamField>
<ParamField path="pool_type">
Pool attributes, determined by the pool&#x27;s factory
</ParamField>
<ParamField path="swap_amount_usd">
Daily swap volume on a pool, in USD
</ParamField>
<ParamField path="tvl_usd">
Total Value Locked on a pool, in USD
</ParamField>
<ParamField path="tvl_eth">
Total Value Locked on a pool, in eth
</ParamField>
<ParamField path="fee_amount_usd">
Daily fees collected on a pool, in USD
</ParamField>

## balancer.bpt_prices 
Balancer Pool Token (BPT) hourly median price by pool on Balancer, an automated portfolio manager and trading platform.


<ParamField path="blockchain">
Blockchain
</ParamField>
<ParamField path="day">
Block date in UTC
</ParamField>
<ParamField path="version">
Version of Balancer where the liquidity pool used in transaction is deployed
</ParamField>
<ParamField path="decimals">
Token decimals. 18, by default, on BPTs
</ParamField>
<ParamField path="contract_address">
zkevm address for the liquidity pool used in transaction
</ParamField>
<ParamField path="bpt_price">
Price of the BPT
</ParamField>

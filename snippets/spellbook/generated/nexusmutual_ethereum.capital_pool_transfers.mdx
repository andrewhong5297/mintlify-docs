## nexusmutual_ethereum.capital_pool_transfers 
Token transfers in the Nexus Mutual Capital Pool

<ParamField path="block_time">
UTC event block time of each transfer
</ParamField>
<ParamField path="block_number">
Block number of each transfer
</ParamField>
<ParamField path="block_date">
UTC event block date of each transfer
</ParamField>
<ParamField path="transfer_type">
Transfer type: in/out (inflow/outflow)
</ParamField>
<ParamField path="symbol">
Token symbol for transferred token
</ParamField>
<ParamField path="amount">
Token amount for transferred token
</ParamField>
<ParamField path="contract_address">
Contract address of the transferred token
</ParamField>
<ParamField path="unique_key">
Generated unique key of the corresponding token transfer
</ParamField>
<ParamField path="tx_hash">
unique transaction hash value tied to each token transfer
</ParamField>
